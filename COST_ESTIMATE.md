### S3 ストレージ試算

| 項目 | 計算 |
|-----|------|
| 想定枚数 | 300枚（100人 x 3枚平均） |
| 1枚のサイズ | 3〜5MB（スマホ写真） |
| 総容量 | 約1〜1.5GB |
| S3無料枠 | 5GB（Standard Storage） |
| 判定 | **無料枠内** |

### Turso 利用量試算

| 項目 | 計算 |
|-----|------|
| Guestレコード | 100件（100人） |
| Sceneレコード | 3〜5件 |
| Photoレコード | 300件 |
| 総データ量 | 数百KB（無視できるサイズ） |
| 読み取り回数 | アルバム表示×数百回 = 数千行読み |
| 書き込み回数 | ニックネーム登録100回 + 写真メタ300回 = 400行書き |
| 判定 | **無料枠の0.01%以下** |

## コスト最適化のポイント

1. **Amplifyのデフォルトドメインを使用** → ドメイン費用¥0
2. **写真保持期間を限定** → イベント後3ヶ月でS3から削除、ローカル保存に切替え
3. **Tursoは利用量が極小** → 無料枠の心配なし
